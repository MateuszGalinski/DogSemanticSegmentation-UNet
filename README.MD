# Overview
Main file with analysis, inference and testing is train.ipynb
.py files are supporting classes for trin.ipynb file

# Dependencies
dependencies are in requirements.txt file and can be installed as follows:
```
pip install -r requirements.txt
```

After that train.ipynb should be runable
